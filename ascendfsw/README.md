# Error Codes

| LEDs (1 = blinking) | Error                                                       |
|---------------------|-------------------------------------------------------------|
| 111                 | Critical Fail                                               |
| 110                 | SD card write fail (could be recovered from, but unlikely)  |
| 101                 | Low sensor count                                            |
| 100                 | Power cycled (determined based off SD card data)            |
| 011                 |                                                             |
| 010                 |                                                             |
| 001                 | None                                                        |
| 000                 | Unused - 1 LED should always be blinking                    |
